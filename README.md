## 🚀 Overview
Welcome to the **E-Commerce Web App**! This is a **full-stack** application built using:

- 🌟 **Frontend:** React.js
- ⚙️ **Backend:** Node.js (Express)
- 🗄️ **Database:** MySQL

This project allows users to browse products, add them to the cart, and securely checkout.
